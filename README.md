# 2025 자율주행 전기차 카메라 영상처리 교육 기록

본 저장소는 2025년 여름방학 동안 진행된 **자율주행 전기차 및 카메라 영상처리 교육과정**의 학습 내용을 정리하고 실습 코드를 기록한 공간입니다.  
주로 **OpenCV**, **Python**, **Git**, **자율주행 제어 이론** 등을 활용하여 실제 자율주행 미니카 및 영상 데이터를 처리하는 프로젝트를 수행하였습니다.

---

## 학습 목표

- OpenCV를 활용한 영상처리 실습
- 자율주행 알고리즘 기초 이해
- Git을 활용해보고 버전관리 해보기
- 실습 결과물을 체계적으로 문서화
